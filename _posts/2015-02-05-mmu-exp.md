arm-linux-ld -Tmmu.lds -o mmu_elf $^

mmu.lds 链接脚本

```
SECTIONS {
  firtst    0x00000000 : { head.o init.o }
  second    0xB0004000 : AT(2048) { leds.o }
}
```

分为两段，第一段将head.S和init.c编译后的文件链接到0x00000000地址
第二段将leds.c文件编译后放到0xB0004000地址
